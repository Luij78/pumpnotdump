# Memecoin Analysis Criteria 📊
*Updated: 2026-01-26 based on Luis's guidance*

## The Core 4 Metrics

### 1. Social Engagement (EXPONENTIAL GROWTH)
- Twitter followers trending UP
- Engagement rate (likes/replies/RTs per post)
- New articles/coverage appearing
- Reddit/Discord/Telegram activity
- **Tool:** Check @handle followers, search for mentions

### 2. Holder Count (INCREASING)
- Total unique holders going UP over time
- Distribution improving (not too concentrated)
- New wallets buying > wallets selling
- **Tool:** Birdeye API, Solscan (need API key)

### 3. Price Action (MOVING UP)
- Consistent uptrend, not just one pump
- Higher lows forming
- Volume supporting the moves
- Already past the $1-2M "fail zone"
- **Tool:** DexScreener, TradingView

### 4. Narrative (RELATABLE)
- Can normies understand it?
- Does it connect to current events?
- Is there a meme/story people can share?
- Cultural relevance matters

## Red Flags to Avoid 🚨
- Token < 24 hours old
- 90%+ held by top 10 wallets
- No social presence at all
- Liquidity < $20K
- Already pumped 1000%+ today
- Buy/sell ratio way too one-sided (>10:1)

## Most Memecoins Fail
- Barely reach $1-2M market cap
- Look for ones that BROKE THROUGH this ceiling
- $10M+ mcap = survived the gauntlet

## Research Checklist
- [ ] Check Twitter follower count + growth
- [ ] Search for recent articles/mentions
- [ ] Check holder count (if available)
- [ ] Verify price is trending up (not just spiked)
- [ ] Evaluate narrative - is it relatable?
- [ ] Confirm past $2M mcap threshold
- [ ] Check buy/sell ratio is healthy
- [ ] Verify liquidity > $30K
