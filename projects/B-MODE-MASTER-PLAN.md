### 4. Flip Calculator

**Logic:**
```
For each new listing in target area:
  zillow_estimate = fetch_zestimate(address)
  list_price = listing.price
  discount = (zillow_estimate - list_price) / zillow_estimate
  
  if discount >= 0.15:  # 15%+ below estimate
    ALERT: "Potential flip deal found"
```

**Alert threshold:** 15% below Zestimate
**Bonus alerts:** 
- Price drops >5%
- DOM >45 days
- Estate/foreclosure keywords
