### APPRAISALS & REGULATION (Sections 1-2)

**1.** Appraisal fees are based on:
- (a) The value of the property
- (b) The time, effort, and expense involved
- (c) A percentage of the selling price
- (d) The lender's requirements
> **Answer: B** — Fees are NEVER based on property value.

**2.** Which board administers Part II of F.S. 475 (appraisals)?
- (a) FREC
- (b) FREAB
- (c) NAR
- (d) USPAP
> **Answer: B** — Florida Real Estate Appraisal Board (FREAB).

**3.** Title XI of FIRREA requires federally-related appraisals to conform to:
- (a) CMA standards
- (b) FREC guidelines
- (c) USPAP
- (d) NAR Code of Ethics
> **Answer: C** — Uniform Standards of Professional Appraisal Practice.

**4.** A real estate licensee may perform an appraisal ONLY if:
- (a) They have a broker's license
- (b) The valuation is not for a federally related transaction
- (c) They have completed 100 hours of appraisal education
- (d) The property value is under $250,000
> **Answer: B** — Licensees can appraise only for non-federal transactions.

**5.** A CMA may NOT be referred to as:
- (a) An estimate of value
- (b) An opinion of value
- (c) An appraisal
- (d) A range of values
> **Answer: C** — Never call a CMA an appraisal.

### TYPES OF VALUE (Section 2)

**6.** The ad valorem value assigned by the county property appraiser is called:
- (a) Market value
- (b) Assessed value
- (c) Investment value
- (d) Insured value
> **Answer: B** — Ad valorem = assessed value for tax purposes.

**7.** Which type of value is specific to a particular investor?
- (a) Market value
- (b) Assessed value
- (c) Investment value
- (d) Liquidation value
> **Answer: C** — Investment value is to a specific individual.

**8.** The most often estimated value in real estate valuation is:
- (a) Investment value
- (b) Insured value
- (c) Market value
- (d) Liquidation value
> **Answer: C** — Market value is the most commonly estimated.

**9.** Market value is:
- (a) Always equal to the selling price
- (b) The amount that SHOULD be paid for a property
- (c) Set by the seller
- (d) Always equal to the assessed value
> **Answer: B** — Market value = what should be paid, not necessarily what is paid.

**10.** The term "price" in real estate represents:
- (a) The appraised value
- (b) The asking amount
- (c) The total amount actually paid by the buyer
- (d) The assessed value
> **Answer: C** — Price = actual amount paid.

### APPRAISAL PRINCIPLES (Section 2)

**11.** A $75,000 kitchen remodel adds $60,000 to home value. This illustrates:
- (a) Progression
- (b) Substitution
- (c) Contribution
- (d) Anticipation
> **Answer: C** — Contribution = value added, not cost spent.

**12.** The smallest house on a block of expensive homes will likely increase in value. This is:
- (a) Regression
- (b) Contribution
- (c) Conformity
- (d) Progression
> **Answer: D** — Progression = worst house on best block goes UP.

**13.** The most expensive house on a block of cheaper homes will likely decrease in value. This is:
- (a) Progression
- (b) Regression
- (c) Substitution
- (d) Conformity
> **Answer: B** — Regression = best house on worst block goes DOWN.

**14.** A buyer won't pay more for a property than the cost of an acceptable substitute. This is:
- (a) Contribution
- (b) Competition
- (c) Substitution
- (d) Anticipation
> **Answer: C** — Substitution is the basis for all appraisal math methods.

**15.** The value of a property today is the sum of its future benefits. This is:
- (a) Anticipation
- (b) Change
- (c) Progression
- (d) Contribution
> **Answer: A** — Anticipation = present value based on future benefits.

**16.** An appraisal is made as of a specific date because:
- (a) Principle of conformity
- (b) Principle of change
- (c) Principle of contribution
- (d) Principle of substitution
> **Answer: B** — Change = market circumstances affect value over time.

**17.** Value is sustained when a property matches other properties in the area. This is:
- (a) Substitution
- (b) Progression
- (c) Conformity
- (d) Competition
> **Answer: C** — Conformity = similar size, style, features sustain value.

### APPRAISAL PROCESS (Section 3)

**18.** The three approaches to value used by appraisers are:
- (a) Cost-depreciation, income, comparable sales
- (b) Market, investment, liquidation
- (c) CMA, BPO, AVM
- (d) Substitution, contribution, anticipation
> **Answer: A** — The three formal appraisal approaches.

**19.** Reconciliation of value indications is accomplished by:
- (a) Averaging the three approach values
- (b) Using the highest value
- (c) Weighing evidence based on appraiser's knowledge and experience
- (d) Using a mathematical formula
> **Answer: C** — NEVER average. Appraiser weighs evidence.

**20.** Which report type is the preference for most residential appraisals?
- (a) Narrative report
- (b) Oral report
- (c) Form report
- (d) Letter report
> **Answer: C** — Form reports are standard for residential.

**21.** Oral appraisal reports are generally given in connection with:
- (a) Residential sales
- (b) Court testimony
- (c) Loan applications
- (d) CMA presentations
> **Answer: B** — Oral reports = court testimony.

**22.** Highest and best use analysis requires the use to be all EXCEPT:
- (a) Legally permissible
- (b) Physically possible
- (c) Previously attempted
- (d) Maximally productive
> **Answer: C** — The four criteria are: legal, physical, financial, maximal. Not "previously attempted."

**23.** Chris inherited property worth $500K as-is. After $150K in improvements, it would be worth $800K. He improves it. This illustrates:
- (a) Contribution
- (b) Highest and best use
- (c) Substitution
- (d) Anticipation
> **Answer: B** — Choosing the use that produces highest value.

### COST-DEPRECIATION APPROACH (Section 4)

**24.** The cost-depreciation formula is:
- (a) Land value + Building cost = Property value
- (b) Building cost - Depreciation + Land value = Property value
- (c) Market value - Depreciation = Property value
- (d) Income / Cap rate = Property value
> **Answer: B** — Cost to build minus depreciation plus land.

**25.** Which cost type is preferred for appraising historic properties?
- (a) Replacement cost
- (b) Reproduction cost
- (c) Market cost
- (d) Liquidation cost
> **Answer: B** — Reproduction cost = exact replica with same materials.

**26.** Replacement cost uses:
- (a) Exact same materials and design
- (b) Modern materials with equal utility
- (c) The lowest available materials
- (d) Only recycled materials
> **Answer: B** — Replacement = comparable building, modern materials.

### THE CMA (Section 5)

**27.** When adjusting comparables in a CMA, you should adjust:
- (a) The subject property
- (b) The comparable property
- (c) Both properties equally
- (d) Neither — use unadjusted values
> **Answer: B** — ALWAYS adjust the comparable, NEVER the subject.

**28.** A comp has 3 bedrooms, subject has 4. Your adjustment:
- (a) Subtract bedroom value from comp
- (b) Add bedroom value to subject
- (c) Add bedroom value to comp
- (d) Subtract bedroom value from subject
> **Answer: C** — Comp lacks a feature → ADD to comp.

**29.** A comp has a pool, subject does not. Your adjustment:
- (a) Add pool value to comp
- (b) Subtract pool value from comp
- (c) Add pool value to subject
- (d) No adjustment needed
> **Answer: B** — Comp has extra feature → SUBTRACT from comp.

**30.** Two identical homes sell, one with a lake view ($375K) and one without ($355K). The lake view adds:
- (a) $375,000
- (b) $30,000
- (c) $20,000
- (d) $355,000
> **Answer: C** — Matched pair: $375K - $355K = $20K.

**31.** The most commonly used method for adjusting square footage differences is:
- (a) Matched pair technique
- (b) Square footage technique
- (c) Cost approach
- (d) Income approach
> **Answer: B** — Square footage technique is most used.

**32.** The most reliable comparable property is the one with:
- (a) The highest adjusted price
- (b) The most adjustments
- (c) The fewest adjustments
- (d) The lowest price
> **Answer: C** — Fewest adjustments = most reliable comp.

**33.** When presenting CMA results to a seller, you should provide:
- (a) An exact price
- (b) The highest possible number
- (c) A range of values
- (d) The assessed value
> **Answer: C** — Give a RANGE, not exact figure.

**34.** Which of these should NOT be used as a comparable in a CMA?
- (a) A home sold 3 months ago in the same neighborhood
- (b) A foreclosure sale from 6 months ago
- (c) A similar home sold 4 months ago nearby
- (d) A home sold 5 months ago, same size and style
> **Answer: B** — Foreclosures are not arm's length transactions.

**35.** A pending sale provides which type of information?
- (a) The actual sales price
- (b) The listing (asking) price only
- (c) The appraised value
- (d) The assessed value
> **Answer: B** — Pending = listing price, not sales price.

**36.** A comparable property that sold 26 months ago should be:
- (a) Used with caution
- (b) Given extra weight
- (c) Not used — too old
- (d) Adjusted for time only
> **Answer: C** — 2+ years is too outdated for reliable CMA.

**37.** CMA Step 1 information gathering should NOT include:
- (a) Number of bedrooms and bathrooms
- (b) Square footage
- (c) Crime rates and ethnic makeup of the area
- (d) Lot size and location
> **Answer: C** — Fair Housing Act prohibits considering race/ethnicity.

### SHORT SALES, REOs, AND BPOs (Section 6)

**38.** A BPO is an estimate of value for a property owned by:
- (a) The seller
- (b) The buyer
- (c) The bank
- (d) The appraiser
> **Answer: C** — BPO = value estimate for bank-owned (REO) property.

**39.** REO stands for:
- (a) Real Estate Option
- (b) Real Estate Owned
- (c) Real Estate Offering
- (d) Real Estate Operation
> **Answer: B** — Real Estate Owned (bank owns it after foreclosure).

**40.** A BPO is sometimes referred to as:
- (a) A CMA
- (b) An automated valuation model (AVM)
- (c) An REO
- (d) An appraisal
> **Answer: B** — BPO = AVM (submitted by computer).

**41.** Who may prepare a BPO in Florida? All EXCEPT:
- (a) Active broker
- (b) Active sales associate
- (c) Student in pre-licensing class
- (d) Active appraisal licensee
> **Answer: C** — Must have an ACTIVE license.

**42.** An outsourcer in the REO context is:
- (a) A real estate agent
- (b) An asset management company handling REO from assignment to sale
- (c) A property inspector
- (d) A title company
> **Answer: B** — Outsourcer = asset management company.

**43.** In a market with many distressed properties, a seller should:
- (a) Price above distressed sales because their property isn't distressed
- (b) Ignore distressed sales in their pricing
- (c) Compete in the market that exists, including distressed properties
- (d) Wait for the market to recover
> **Answer: C** — Must compete in the actual market.

**44.** BPO factors that affect property price include all EXCEPT:
- (a) Values of similar surrounding properties
- (b) Sales trends in the neighborhood
- (c) The owner's original purchase price
- (d) Amount of repair needed
> **Answer: C** — Original purchase price is irrelevant to current value.

**45.** A BPO is:
- (a) More thorough than an appraisal
- (b) Less thorough than an appraisal but more than a CMA
- (c) The same as a CMA
- (d) Only done by appraisers
> **Answer: B** — CMA < BPO < Appraisal in thoroughness.

8. Fewest adjustments = most reliable comp

11. Reconciliation = appraiser judgment, NEVER averaging
