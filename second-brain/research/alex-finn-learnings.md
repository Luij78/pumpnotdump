## 🔧 Technical Tips from Alex

### Claude Code Session Management
- `claude --res` - Resume past sessions
- `/context` - View what's in context window
- `/clear` - Clear message history (reduces hallucinations)
- `/stats` - Track token usage
- `/rename [name]` - Name sessions for easy identification
- **Double Escape** - Rewind recent mistakes
- `Ctrl+S` - Stash/save prompts temporarily

### "Ultra Think" for Complex Tasks
Include "Ultra Think" in prompts for:
- Complex bug fixes
- Building foundational app architecture
- Tasks requiring deep reasoning
Uses more tokens but significantly better results.

### YOLO Mode
`claude-dangerously-skip-permissions` - Auto-approves all actions
- **Warning:** Only for experienced users confident in prompting
- Useful for trusted repetitive workflows

### Plugins to Check Out
- Claude Design Skill - Better UI design
- Front-end Design Plugin
- GitHub, Figma, Linear, Notion integrations
