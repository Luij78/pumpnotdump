## 🏠 Listing Descriptions

### Standard Listing Description
```
Write a compelling MLS listing description for this property:

Address: [ADDRESS]
Beds/Baths: [X] bed / [X] bath
Sqft: [X]
Year Built: [YEAR]
Key Features: [LIST 3-5 FEATURES]
Neighborhood: [NEIGHBORHOOD NAME]
Price: $[PRICE]

Style: Professional but warm. Highlight lifestyle benefits, not just features. 
Length: 200-250 words. Include a strong opening hook.
```

### Luxury Listing
```
Write an elegant, sophisticated listing description for this luxury property:

Address: [ADDRESS]
Beds/Baths: [X] bed / [X] bath
Sqft: [X]
Key Features: [LIST 5-7 PREMIUM FEATURES]
Views/Location: [DESCRIBE]
Price: $[PRICE]

Style: Evocative, lifestyle-focused, aspirational. Use sensory language.
Length: 300-350 words.
```

### Quick Social Media Version
```
Turn this listing into a punchy social media post:

[PASTE FULL LISTING DESCRIPTION]

Make it: Short (under 100 words), attention-grabbing, include a call to action.
Include relevant emojis.
```

## 📧 Email Templates

### First Contact (Buyer Lead)
```
Write a friendly first-contact email to a buyer lead:

Lead name: [NAME]
How they found me: [ZILLOW/REFERRAL/OPEN HOUSE/ETC]
Property they inquired about: [ADDRESS or "general search"]
Their situation (if known): [FIRST TIME BUYER/RELOCATING/ETC]

Tone: Warm, professional, helpful. Not salesy.
Goal: Get them to respond or book a call.
Length: 100-150 words.
```

### First Contact (Seller Lead)
```
Write a first-contact email to a homeowner who requested a home valuation:

Lead name: [NAME]
Property address: [ADDRESS]
Timeline (if known): [ASAP/6 MONTHS/JUST CURIOUS]

Tone: Helpful, knowledgeable, not pushy.
Goal: Book a call to discuss their home's value.
Length: 100-150 words.
```

### Follow-Up (No Response)
```
Write a friendly follow-up email. The lead hasn't responded to my first email.

Lead name: [NAME]
Days since first email: [X]
What I offered: [CALL/HOME SEARCH/VALUATION]

Tone: Light, helpful, no guilt-tripping.
Length: 50-75 words.
Add a simple question to encourage response.
```

### Re-Engagement (Cold Lead)
```
Write an email to re-engage a lead I haven't talked to in 3+ months:

Lead name: [NAME]
Original interest: [BUYING/SELLING]
Last interaction: [DESCRIBE]

Tone: Casual, checking-in, no pressure.
Mention something valuable (market update, new listings, rate changes).
Length: 75-100 words.
```

## 🎯 Objection Handlers

### "The price is too high"
```
A buyer says the listing price seems too high. Write a response that:
- Acknowledges their concern
- Explains market factors
- Offers to show comparable properties
- Keeps them engaged

Property price: $[PRICE]
Market context: [HOT/BALANCED/COOLING]
Tone: Understanding, knowledgeable, not defensive.
```

### "I want to think about it"
```
A buyer says they need time to think after a showing. Write a response that:
- Respects their process
- Gently creates urgency if appropriate
- Offers helpful next steps
- Leaves door open

Property situation: [HOT MARKET/MULTIPLE OFFERS/PRICE DROP COMING/ETC]
Tone: Supportive, not pushy.
```

### "I'm working with another agent"
```
A lead says they're already working with another agent. Write a response that:
- Is professional and gracious
- Leaves a positive impression
- Opens door for future contact

Tone: Respectful, not desperate. Short.
```

### "I'll list with a family member/friend"
```
A potential seller says they plan to list with a friend or family member. Write a response that:
- Is gracious and understanding
- Subtly mentions benefits of working with a dedicated professional
- Offers to be a resource if things change

Tone: Friendly, not competitive.
```

Make it educational and shareable. Under 100 words.

Authentic, relatable, human. Not salesy. Under 100 words.

## 🛠️ Tips for Best Results

### Make It Your Own
- Add your name and contact info
- Adjust tone to match your personality
- Include local references and knowledge

### Iterate
- If the first result isn't perfect, say "Make it more [casual/professional/urgent]"
- Ask for 3 versions and pick the best

### Save Your Best Prompts
- When something works, save it to your prompt library
- Update prompts with what you learn
