# AI Quick Start — Service Package Details

**Service Name:** AI Quick Start for Real Estate Agents  
**Price:** $750 setup + $150/mo optional maintenance  
**Delivery Time:** 5-7 days from kickoff

---

## What's Included

### 1. Discovery Call (30 min)
- Understand their current workflow
- Identify biggest pain points
- Map out what AI can solve for them
- Set expectations and timeline

### 2. ChatGPT Prompt Library (Customized)
**Deliverable:** Google Doc or Notion page with 20+ ready-to-use prompts

Includes:
- Listing description generator (plug in property details, get polished copy)
- Buyer/seller email templates (first contact, follow-up, nurture)
- Objection handlers (price too high, want to think about it, competitor agent)
- Social media captions (just listed, just sold, market update)
- Text message templates (appointment confirmation, check-in, re-engagement)
- Neighborhood description generator
- Open house follow-up sequences

### 3. Automated Follow-Up System
**Tool:** Zapier + Google Sheets (or their existing CRM if compatible)

Includes:
- Lead capture → auto-assignment workflow
- Day 1/3/7/14/30 follow-up email sequence
- Missed call text-back automation
- Lead tagging and status tracking
- Simple dashboard to see all leads at a glance

### 4. Lead Qualification Chatbot
**Tool:** ChatGPT custom GPT or simple form flow

What it does:
- Asks qualifying questions automatically
- Categorizes leads (hot/warm/cold)
- Provides summary for agent before first call
- Reduces time wasted on unqualified leads

### 5. Training Session (60 min, recorded)
- Walk through every system I set up
- Show them how to use the prompts
- Demonstrate the automation
- Answer questions live
- Record for future reference

### 6. 30-Day Support
- Text/email support for questions
- Quick fixes and tweaks
- Help troubleshooting issues
- Additional prompts as needed
